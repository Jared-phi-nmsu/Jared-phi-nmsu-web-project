Project 3

NOT Gate AND Gate NAND Gate OR Gate NOR Gate XOR Gate XNOR Gate

This is my first website and It was a challenge to not use CSS. The goal of this website is to demonstrate all the logic gates represented in Minecraft. I have used a lot of images all of which are in the open domain or I screen shot my self. I enjoyed the challenge and it made me appreciate old website design.

My goal for project 3 was to remake project 2 with out using HTML style but by using css. To do so we can separate the project into base HTML elements. We can target these elements in CSS to then apply their style settings. For my project I have a main page and seven sub pages, I designed the CSS document to target the sub pages and some elements on the main page. I then assigned classes to some specific elements and adjusted as needed. I was almost able to copy and paste some HTML style sections into their corresponding css sections. During project 2, I found keeping my pages uniform really tedious and time consuming. Using css allowed me to make rapid adjustments saving a lot of time, instead of finding a change then implementing it 7 times over 7 files. I find something to change or improve and I can apply it instantly over my entire base. Further, I added some hover animations to the li element, I remade the outdated marquee element with a div container and css animations. Doing so I was able to get the animation to pause on hover as well as display an translation. I added style to all tables.
